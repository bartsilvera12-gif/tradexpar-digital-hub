# Tradexpar Digital Hub

Tienda y panel admin (Vite, React, Supabase). Requisitos y variables: ver [`.env.example`](.env.example).

- **Desarrollo:** `npm install` → `npm run dev`
- **Producción:** `npm run build` (definí `VITE_PUBLIC_SITE_URL` sin barra final para metadatos Open Graph con URL absoluta en el artículo `og:image`)
